LDA

Different from PCA, LDA (Linear Discriminant Analysis) wants to make data with same label clustered together in the low dimension space. LDA assumes that the original data is classified based on the mean value, and different types of value have the same variance. Thus, LDA performs better when the original data is well separated by the mean value.

Assume we have,

D={(x1,y1),(x2,y2),…,(xn,yn)}\mathcal{D} = \{(x_1, y_1), (x_2, y_2), \ldots, (x_n, y_n)\}D={(x1​,y1​),(x2​,y2​),…,(xn​,yn​)}

yi∈Yy_i \in \mathbb{Y}yi​∈Y where Y\mathbb{Y}Y is the set of all possible labels.

We calculate the mean and variance for each label j∈Yj \in \mathbb{Y}j∈Y

μj=1nj∑i=1nxiI(yi=j)\mu_j = \frac{1}{n_j} \sum_{i=1}^{n} x_i \mathbb{I}(y_i = j)μj​=nj​1​i=1∑n​xi​I(yi​=j) σj2=1nj∑i=1n(xi−μj)(xi−μj)TI(yi=j)\sigma_j^2 = \frac{1}{n_j} \sum_{i=1}^{n} (x_i - \mu_j)(x_i - \mu_j)^T \mathbb{I}(y_i = j)σj2​=nj​1​i=1∑n​(xi​−μj​)(xi​−μj​)TI(yi​=j)

Please note that μj\mu_jμj​ is the mean vector and σj2\sigma_j^2σj2​ is the covariance matrix.

We also use a the following formula to measure the within-class variance,

Σw2=1n∑j=1∣Y∣njσj2\Sigma_w^2 = \frac{1}{n} \sum_{j=1}^{|\mathbb{Y}|} n_j \sigma_j^2Σw2​=n1​j=1∑∣Y∣​nj​σj2​

And we use the following formula to measure the between-class variance,

Σb2=1n∑j=1∣Y∣nj(μj−x‾)(μj−x‾)T\Sigma_b^2 = \frac{1}{n} \sum_{j=1}^{ | \mathbb{Y} | } n_j (\mu_j - \overline{x}) (\mu_j - \overline{x})^TΣb2​=n1​j=1∑∣Y∣​nj​(μj​−x)(μj​−x)T

Where x‾\overline{x}x is the mean of all vectors.

The variance of the whole dataset is,

Σ2=1n∑i=1n(xi−xˉ)(xi−xˉ)T\Sigma^2 = \frac{1}{n} \sum_{i=1}^n (x_i - \bar{x}) (x_i - \bar{x})^TΣ2=n1​i=1∑n​(xi​−xˉ)(xi​−xˉ)T

We can prove that,

Σ2=Σw2+Σb2\Sigma^2 = \Sigma_w^2 + \Sigma_b^2Σ2=Σw2​+Σb2​

We use the following property,

σ2(x)=E(xxT)−E(x)E(x)T\sigma^2(x) = \mathbb{E}(xx^T) - \mathbb{E}(x)\mathbb{E}(x)^Tσ2(x)=E(xxT)−E(x)E(x)T

Thus,

Σw2+Σb2=1n∑i=1n((xi−μj)(xi−μj)T+(μj−x‾)(μj−x‾)T)=1n∑i=1n(xixiT+x‾  x‾T−μjxiT−xiμjT−μjx‾T−x‾μjT+2μjμjT)=Ei(xixiT)+x‾  x‾T−Ei(xiμjT)−Ei(μjxiT)−Ei(μjx‾T)−Ei(x‾μjT)+2Ei(μjμjT)\Sigma_w^2 + \Sigma_b^2 \\ = \frac{1}{n} \sum_{i=1}^{n} ((x_i - \mu_j) (x_i - \mu_j)^T + (\mu_j - \overline{x}) (\mu_j - \overline{x})^T)\\ = \frac{1}{n} \sum_{i=1}^{n} (x_ix_i^T + \overline{x} \; \overline{x}^T - \mu_j x_i^T - x_i \mu_j^T - \mu_j \overline{x}^T - \overline{x} \mu_j^T +2 \mu_j\mu_j^T)\\ = \mathbb{E_i}(x_ix_i^T) + \overline{x} \; \overline{x}^T - \mathbb{E_i}(x_i \mu_j^T) - \mathbb{E_i}(\mu_j x_i^T) - \mathbb{E_i}(\mu_j \overline{x}^T) - \mathbb{E_i}(\overline{x} \mu_j^T) + 2 \mathbb{E_i}(\mu_j\mu_j^T)\\Σw2​+Σb2​=n1​i=1∑n​((xi​−μj​)(xi​−μj​)T+(μj​−x)(μj​−x)T)=n1​i=1∑n​(xi​xiT​+xxT−μj​xiT​−xi​μjT​−μj​xT−xμjT​+2μj​μjT​)=Ei​(xi​xiT​)+xxT−Ei​(xi​μjT​)−Ei​(μj​xiT​)−Ei​(μj​xT)−Ei​(xμjT​)+2Ei​(μj​μjT​)

Simplify term by term:

1n∑i=1nxixiT=E(xxT)\frac{1}{n} \sum_{i=1}^n x_i x_i^T = \mathbb{E}(x x^T)n1​i=1∑n​xi​xiT​=E(xxT)

The terms −xiμjT-x_i \mu_j^T−xi​μjT​ and −μjxiT-\mu_j x_i^T−μj​xiT​ combine to:

−1n∑jnj(μjμjT+μjμjT)=−2n∑jnjμjμjT-\frac{1}{n} \sum_{j} n_j (\mu_j \mu_j^T + \mu_j \mu_j^T) = -\frac{2}{n} \sum_{j} n_j \mu_j \mu_j^T−n1​j∑​nj​(μj​μjT​+μj​μjT​)=−n2​j∑​nj​μj​μjT​

This is valid because, for each cluster jjj, μj\mu_jμj​ is a constant, and its indices Gj\mathbb{G}_jGj​,

∑i∈Gj−xiμjT=−njμjμjT\sum_{i\in\mathbb{G}_j} -x_i \mu_j^T = -n_j \mu_j \mu_j^Ti∈Gj​∑​−xi​μjT​=−nj​μj​μjT​

The same goes for −μjxiT-\mu_j x_i^T−μj​xiT​.

The terms −μjx‾T-\mu_j \overline{x}^T−μj​xT and −x‾μjT-\overline{x} \mu_j^T−xμjT​ combine to:

−1n∑jnj(μjx‾T+x‾μjT)=−2x‾x‾T-\frac{1}{n} \sum_{j} n_j (\mu_j \overline{x}^T + \overline{x} \mu_j^T) = -2 \overline{x} \overline{x}^T−n1​j∑​nj​(μj​xT+xμjT​)=−2xxT

The terms μjμjT\mu_j \mu_j^Tμj​μjT​ and x‾x‾T\overline{x} \overline{x}^TxxT simplify using the identity ∑jnjμj=nx‾\sum_{j} n_j \mu_j = n \overline{x}∑j​nj​μj​=nx:

1n(2∑jnjμjμjT+nx‾x‾T)=2E(μjμjT)+x‾x‾T\frac{1}{n} \left( 2 \sum_{j} n_j \mu_j \mu_j^T + n \overline{x} \overline{x}^T \right) = 2 \mathbb{E}(\mu_j \mu_j^T) + \overline{x} \overline{x}^Tn1​(2j∑​nj​μj​μjT​+nxxT)=2E(μj​μjT​)+xxT

In all,

Σw2+Σb2=E(xxT)−2E(μjμjT)+2E(μjμjT)−2x‾x‾T+x‾x‾T=E(xxT)−x‾x‾T=Σ2\Sigma_w^2 + \Sigma_b^2 \\ = \mathbb{E}(x x^T) - 2 \mathbb{E}(\mu_j \mu_j^T) + 2 \mathbb{E}(\mu_j \mu_j^T) - 2 \overline{x} \overline{x}^T + \overline{x} \overline{x}^T \\ = \mathbb{E}(x x^T) - \overline{x} \overline{x}^T \\ = \Sigma^2Σw2​+Σb2​=E(xxT)−2E(μj​μjT​)+2E(μj​μjT​)−2xxT+xxT=E(xxT)−xxT=Σ2

However, please pay attention that, in all the calculation above, the presume every element is already labeled. Thus there is no variable. However, let's back to the main track, if we project the original data onto a low-dimensional space, that is, a non-square projection matrix WTW^TWT that could maximize the between-class variance after projection, and minimize the within-class variance.

For convenience, we use WTW^TWT so that every column vector is a new basis.

That is,

x′=WTxx' = W^T xx′=WTx μj′=WTμj\mu_j' = W^T \mu_jμj′​=WTμj​ σj′2=WTσj2W\sigma_j'^2 = W^T \sigma_j^2 Wσj′2​=WTσj2​W Σb′2=WTΣb2W\Sigma_b'^2 = W^T \Sigma_b^2 WΣb′2​=WTΣb2​W Σw′2=WTΣw2W\Sigma_w'^2 = W^T \Sigma_w^2 WΣw′2​=WTΣw2​W

We would like to maximize the Σb′2\Sigma_b'^2Σb′2​, and minimize the Σw′2\Sigma_w'^2Σw′2​. However, they are matrices, not scalars, so we cannot directly compare them. LDA choose the following target function,

J(WT)=∣∣Σb′2−1Σw′2∣∣FJ(W^T) = ||{\Sigma_b'^2}^{-1}\Sigma_w'^2||_{F}J(WT)=∣∣Σb′2​−1Σw′2​∣∣F​
tip

The ∣∣M∣∣F||M||_{F}∣∣M∣∣F​ is the Frobenius norm, defined as,

∣∣M∣∣F=∑i,jMi,j2||M||_{F} = \sqrt{\sum_{i,j} M_{i,j}^2}∣∣M∣∣F​=i,j∑​Mi,j2​​

Which, if we define λi\lambda_iλi​ as the eigenvalues of MMM, also equals,

∣∣M∣∣F=∑iλi2||M||_{F} = \sqrt{\sum_{i} \lambda_i^2}∣∣M∣∣F​=i∑​λi2​​

Another point to consider is that, we must restrain all column vector (the new basis) to have a unit norm. Otherwise, the result can be arbitrarily small or large because you can add a scale factor to the row vector.

That is to say,

WTW=1W^T W = 1WTW=1

We suppose, www is a unit eigenvector, so,

wTΣb′2−1Σw′2w=λ2=wTWT(Σb2−1Σw2)Ww=(WTw)T(Σb2−1Σw2)(WTw)w^T {\Sigma_b'^2}^{-1}\Sigma_w'^2 w = \lambda^2 \\ = w^T W^T ({\Sigma_b^2}^{-1}\Sigma_w^2) W w\\ = (W^T w)^T ({\Sigma_b^2}^{-1}\Sigma_w^2) (W^T w) \\wTΣb′2​−1Σw′2​w=λ2=wTWT(Σb2​−1Σw2​)Ww=(WTw)T(Σb2​−1Σw2​)(WTw)

That is to say, WTwW^T wWTw is the eigenvector of (Σb2−1Σw2)({\Sigma_b^2}^{-1}\Sigma_w^2)(Σb2​−1Σw2​).

So,

J(WT)=∣∣Σb′2−1Σw′2∣∣F=∑i∈Lλi2J(W^T) = ||{\Sigma_b'^2}^{-1}\Sigma_w'^2||_{F} = \sqrt{\sum_{i \in \mathbb{L}} \lambda_i^2}J(WT)=∣∣Σb′2​−1Σw′2​∣∣F​=i∈L∑​λi2​​

Where L\mathbb{L}L is a set of eigenvalues chosen from Σb2−1Σw2{\Sigma_b^2}^{-1}\Sigma_w^2Σb2​−1Σw2​. ∣L∣|\mathbb{L}|∣L∣ equals to the rank of WWW.

So obviously, if we want to minimize J(WT)J(W^T)J(WT), we simply use the smallest eigenvalues of Σb2−1Σw2{\Sigma_b^2}^{-1}\Sigma_w^2Σb2​−1Σw2​.

And because www is an unit vector, we can always use, wi=eiw_i = e_iwi​=ei​. And thus, since WTwW^T wWTw is the eignenvector of Σb2−1Σw2{\Sigma_b^2}^{-1}\Sigma_w^2Σb2​−1Σw2​, WTW^TWT has each of its column vector as an eignenvector of Σb2−1Σw2{\Sigma_b^2}^{-1}\Sigma_w^2Σb2​−1Σw2​.

note

That was a long proof. But the result is simple- we find kkk smallest eigenvalues of Σb2−1Σw2{\Sigma_b^2}^{-1}\Sigma_w^2Σb2​−1Σw2​, stack them to get WTW^TWT. And after performing,

X′=WTXX' = W^T XX′=WTX

We get the LDA result.
