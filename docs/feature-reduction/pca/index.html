PCA

PCA means principal component analysis. It is a technique used to reduce the dimensionality of the dataset.

The idea is very simple, consider a dataset matrix,

X=(x1,x2,…,xn)TX = (x_1, x_2, \ldots, x_n)^TX=(x1​,x2​,…,xn​)T

We would like to retain the dimension that captures the most variance in the dataset.

Consider the covariance matrix, C=(X−X‾)T(X−X‾)C = (X - \overline{X})^T(X - \overline{X})C=(X−X)T(X−X) where X‾=(x‾,x‾,…,x‾)T\overline{X} = (\overline{x}, \overline{x}, \ldots, \overline{x})^TX=(x,x,…,x)T. Because we want maximum variance for each dimension, we can perform an eigendecomposition of the covariance matrix.

C=VΛVTC = V \Lambda V^TC=VΛVT

We will use VVV as the new basis for each xxx data vector. So their variance can be measured by their eigenvalues. Those with higher eigenvalues are the principal components.

We can set a threshold or set a fixed number of principal components to keep. Then we can finish the feature reduction.

info

The above process can also be seen as SVD on the dataset matrix XXX.

From the SVD perspective, the principal components are the left singular vectors of XXX. The eigenvalues of the covariance matrix are the square of the singular values of XXX. We look for the singular vectors that has the largest absolute singular values, then cast the data into the reduced singular vector space.

tip

Singular Value Decomposition refers to decomposing a matrix into,

X=UΣVTX = U \Sigma V^TX=UΣVT

Where UUU and VVV are orthogonal matrices and Σ\SigmaΣ is a diagonal matrix. This is a natural extension of the eigendecomposition of a square matrix.

UUU and VVV are the left and right singular vectors of XXX respectively. Σ\SigmaΣ is the singular values of XXX. For a square matrix, the singular values are the eigenvalues.

UUU is responsible for rotating the vector into a proper basis, then SigmaSigmaSigma brings the space into a another dimension while performing a stretch, then VVV rotates the target space.

To calculate the SVD,

XTX=VΣ2VTX^T X = V \Sigma^2 V^TXTX=VΣ2VT

Because XTXX^T XXTX is a square matrix, it has an eigendecomposition. The eigenvectors of XTXX^T XXTX are the right singular vectors of XXX. The eigenvalues of XTXX^T XXTX are the square of the singular values of XXX.

Similarly,

XXT=UΣ2UTX X^T = U \Sigma^2 U^TXXT=UΣ2UT

Thus we can calculate the singular values and vectors of XXX.
